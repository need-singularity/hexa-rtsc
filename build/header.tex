%     TeX Live distributions.
%   - amsmath / amssymb / unicode-math — closed-form derivations need
%     ⊕ ⊗ σ τ φ etc. as Unicode-direct, not legacy macros.
%   - hyperref colored links — cross-reference between roadmap §A.4
%     falsifier rows and pillar §8.x derivations is heavy.
